# Algorytmy w C++

Zbiór klasycznych algorytmów zaimplementowanych w języku **C++**.  
Repozytorium powstało jako forma nauki i utrwalenia podstawowych koncepcji programowania imperatywnego, strukturalnego i analizy złożoności.

---

## Cel projektu
- Utrwalenie podstaw algorytmiki i analizy złożoności (czasowej i pamięciowej)
- Ćwiczenie poprawnej implementacji w języku C++
- Budowa solidnego fundamentu pod dalszą naukę programowania (np. struktur danych, programowania obiektowego)

---

## Zawartość

Każdy plik zawiera:
- kompletny, samodzielny program w C++,
- komentarze objaśniające działanie algorytmu,
- testy działania (dla różnych rozmiarów danych),
- analizę liczby operacji (porównań, przestawień itp.).
